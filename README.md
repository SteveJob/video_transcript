# Video Transcript Skill

A practical skill that converts one video/audio input into three outputs in one run:

1. Full transcript (`.txt`)
2. Podcast script summary (`_播客脚本.txt`)
3. Podcast audio (`_播客.mp3`, normal speed by default)

This skill does not require an OpenAI API key.

## AI Quick Install

Copy this sentence to Claude Code / Codex:

```text
Please install the Video Transcript skill from git@github.com:SteveJob/video_transcript.git into my skills directory, install required dependencies, and verify the pipeline script can run.
```

Use the skill:

```text
/video-transcript convert <video_or_audio_path>
```

## Local Use

```bash
git clone git@github.com:SteveJob/video_transcript.git
cd video_transcript
python3 -m venv .venv
.venv/bin/python -m pip install openai-whisper edge-tts

.venv/bin/python video_transcript/scripts/video_transcript_podcast.py \
  --input "/absolute/path/to/input.mp4" \
  --transcript-only
```

Then let Claude Code / Codex generate `<basename>_播客脚本.txt` from transcript in current conversation context, and run:

```bash
.venv/bin/python video_transcript/scripts/video_transcript_podcast.py \
  --transcript-file "/absolute/path/to/<basename>.txt" \
  --podcast-script-file "/absolute/path/to/<basename>_播客脚本.txt"
```

## Output Files

For input `example.mp4`, the script writes:

- `example.txt`
- `example_播客脚本.txt`
- `example_播客.mp3`

Default output directory is the same folder as the input file.

## Feature Design

- Input support:
  - Common video/audio formats (e.g. `.mp4`, `.webm`, `.mp3`)
  - Existing transcript mode via `--transcript-file`
- Transcription:
  - Uses `openai-whisper` (`base` model by default)
  - Language default is Chinese (`--language zh`)
- Podcast script generation:
  - Recommended: generated by Codex model context from transcript
  - Script supports external podcast script input via file/text
- Podcast audio generation:
  - Uses `edge-tts` voice synthesis
  - Keeps normal speed by default (`--rate +0%`)
  - Voice is configurable via `--voice`

## Useful Options

```bash
.venv/bin/python video_transcript/scripts/video_transcript_podcast.py \
  --input "/absolute/path/to/input.mp4" \
  --output-dir "/absolute/path/to/output" \
  --model base \
  --language zh \
  --voice zh-CN-XiaoxiaoNeural \
  --rate +0%
```

Or skip transcription and provide external script:

```bash
.venv/bin/python video_transcript/scripts/video_transcript_podcast.py \
  --transcript-file "/absolute/path/to/existing_transcript.txt" \
  --podcast-script-file "/absolute/path/to/podcast_script.txt"
```
